## Free & Open Source Web Scraping/Searching Tools (No Paid API Key Needed)

### Web Search
- **[pskill9/web-search](https://github.com/pskill9/web-search)**: Enables free web searching using Google search results, with no API keys required.
- **[Aas-ee/open-webSearch](https://github.com/Aas-ee/open-webSearch)**: Web search using free multi-engine search (Bing, Baidu, DuckDuckGo, Brave, Exa, CSDN) - NO API KEYS REQUIRED.
- **[nickclyde/duckduckgo-mcp-server](https://github.com/nickclyde/duckduckgo-mcp-server)**: Web search using DuckDuckGo.
- **[SecretiveShell/MCP-searxng](https://github.com/SecretiveShell/MCP-searxng)**: Connects to SearXNG instances (open source meta-search engine).

### Web Scraping & Browser Automation
- **[modelcontextprotocol/server-puppeteer](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer)**: Official Puppeteer server for browser automation and web scraping.
- **[modelcontextprotocol/server-fetch](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)**: Official server for efficient web content fetching and processing.
- **[jae-jae/fetcher-mcp](https://github.com/jae-jae/fetcher-mcp)**: Fetches web page content using Playwright headless browser, supports JS rendering and Markdown output.
- **[just-every/mcp-read-website-fast](https://github.com/just-every/mcp-read-website-fast)**: Fast, token-efficient web content extraction to Markdown.
- **[browsermcp/mcp](https://github.com/browsermcp/mcp)**: Automate your local Chrome browser.

## Other Free & Open Source Tools (Categorized)

### Command Line & Terminal
- **[g0t4/mcp-server-commands](https://github.com/g0t4/mcp-server-commands)**: Run any command with `run_command` and `run_script` tools.
- **[misiektoja/kill-process-mcp](https://github.com/misiektoja/kill-process-mcp)**: List and terminate OS processes via natural language.
- **[tufantunc/ssh-mcp](https://github.com/tufantunc/ssh-mcp)**: Securely execute remote shell commands via SSH.
- **[wonderwhy-er/DesktopCommanderMCP](https://github.com/wonderwhy-er/DesktopCommanderMCP)**: Manage/execute programs and read/write/search/edit code and text files.

### Databases (Local/Self-Hosted)
- **[modelcontextprotocol/server-postgres](https://github.com/modelcontextprotocol/servers/tree/main/src/postgres)**: Official PostgreSQL integration.
- **[modelcontextprotocol/server-sqlite](https://github.com/modelcontextprotocol/servers/tree/main/src/sqlite)**: Official SQLite database interaction.
- **[benborla29/mcp-server-mysql](https://github.com/benborla/mcp-server-mysql)**: MySQL database integration.
- **[amineelkouhen/mcp-cockroachdb](https://github.com/amineelkouhen/mcp-cockroachdb)**: Manage and query CockroachDB.

### File Systems & Local Storage
- **[modelcontextprotocol/server-filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)**: Direct local file system access.
- **[mark3labs/mcp-filesystem-server](https://github.com/mark3labs/mcp-filesystem-server)**: Golang implementation for local file system access.
- **[microsoft/markitdown](https://github.com/microsoft/markitdown/tree/main/packages/markitdown-mcp)**: Converts many file formats (local or remote) to Markdown.
- **[mamertofabian/mcp-everything-search](https://github.com/mamertofabian/mcp-everything-search)**: Fast Windows file search using Everything SDK.

### Developer Tools
- **[axliupore/mcp-code-runner](https://github.com/axliupore/mcp-code-runner)**: Run code locally via Docker, supporting multiple languages.
- **[azer/react-analyzer-mcp](https://github.com/azer/react-analyzer-mcp)**: Analyze React code locally.
- **[cyberchitta/llm-context.py](https://github.com/cyberchitta/llm-context.py)**: Share code context with LLMs via MCP or clipboard.

### Version Control
- **[modelcontextprotocol/server-git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)**: Direct Git repository operations (read, search, analyze local repos).
- **[adhikasp/mcp-git-ingest](https://github.com/adhikasp/mcp-git-ingest)**: Read and analyze GitHub repositories.
- **[gitea/gitea-mcp](https://gitea.com/gitea/gitea-mcp)**: Interact with Gitea instances (self-hosted Git service).
